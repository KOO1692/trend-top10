4사용 안내

사용 안내 / 주의사항

  • 이 결과는 네이버 데이터랩(시장현황)과 네이버 쇼핑 검색 API(상품 검색)가 반환한 데이터입니다.
  • 구매순위, 판매순위, 네이버 공식 인기순위가 아닙니다.
  • 표의 검색순위는 네이버 쇼핑 검색 API가 반환한 순서이고, 가격순위는 현재 화면에 보이는 결과 안에서 최저가 기준으로 매긴 참고 순위입니다.
  • 시장현황은 설정한 관심 키워드 10개 기준이며, 상품 검색의 기본 정렬은 정확도순(sort=sim)입니다.
  • 개인 참고용 대시보드입니다.
  • API 호출이 실패하면 예시 데이터가 표시될 수 있습니다.
  • 최근 검색어는 이 브라우저에만 저장되며, 네이버 API 응답 데이터는 저장하지 않습니다.
  • Render 무료 플랜에서는 오랜만에 접속할 때 첫 로딩이 느릴 수 있습니다.

운영 점검 링크

아래 링크를 누르면 새 탭에서 서버 상태를 직접 확인할 수 있어요.

  • /api/health 서버가 켜져 있는지 확인
  • /api/naver-ready 네이버 API 환경변수가 준비됐는지 확인
  • /api/market-status 시장현황(관심 키워드) 데이터 응답 확인
  • /api/naver-shopping-search 상품 검색 결과 데이터 응답 확인

🔒 이 링크들은 서버 상태나 데이터 응답만 보여주며, API 키(Client ID)나 Secret 값은 어떤 경우에도 표시하지 않습니다.

네이버 쇼핑 검색 결과 TOP10

네이버 쇼핑 검색 API로 확인하는 개인 참고용 대시보드

ℹ️ 네이버 쇼핑 검색 API가 돌려주는 검색 결과를 정확도순(sort=sim)으로 보여주는 개인 참고용 화면이에요. 구매순위·판매순위·네이버 공식 인기순위가 아니며, API 연결에 실패하면 예시 데이터가 표시될 수 있어요.

⚠️ 서버 연결 실패: 화면에는 이전에 불러온 결과(또는 예시 데이터)가 그대로 표시되고 있습니다. PowerShell에서 node server.js를 실행했는지 확인해 주세요.

ℹ️ 네이버 쇼핑 검색 API 호출에 실패했습니다. 화면에는 이전 결과(또는 예시 데이터)가 그대로 표시되고 있습니다. 잠시 후 다시 시도해 주세요.

4. 사용 안내

사용 안내 / 주의사항

  • 이 결과는 네이버 쇼핑 검색 API가 반환한 검색 결과입니다.
  • 구매순위, 판매순위, 네이버 공식 인기순위가 아닙니다.
  • 기본 정렬은 정확도순(sort=sim)입니다.
  • 개인 참고용 대시보드입니다.
  • API 호출이 실패하면 예시 데이터가 표시될 수 있습니다.
  • 최근 검색어는 이 브라우저에만 저장되며, 네이버 API 응답 데이터는 저장하지 않습니다.
  • Render 무료 플랜에서는 오랜만에 접속할 때 첫 로딩이 느릴 수 있습니다.

운영 점검 링크

아래 링크를 누르면 새 탭에서 서버 상태를 직접 확인할 수 있어요.

  • /api/health 서버가 켜져 있는지 확인
  • /api/naver-ready 네이버 API 환경변수가 준비됐는지 확인
  • /api/trends (과거 버전용) 후보 키워드 비교 데이터 응답 확인

🔒 이 링크들은 서버 상태나 데이터 응답만 보여주며, API 키(Client ID)나 Secret 값은 어떤 경우에도 표시하지 않습니다.
